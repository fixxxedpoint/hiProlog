# hiProlog
A toy implementation of a Prolog's interpreter. No cut operator, no negation, just Horn clauses.
## Getting Started
### Prerequisites
Tools required to build this project:
* stack <http://haskellstack.org>
### Build Instructions
```console
stack build
```
### Executing Tests
```console
stack test
```
### Main Executable
```console
stack exec hiProlog-exe
```
